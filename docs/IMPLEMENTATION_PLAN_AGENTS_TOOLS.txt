================================================================================
IMPLEMENTATION PLAN — AGENTS, SKILLS, TOOLS & OPERABILITY
================================================================================
Purpose
  Turn the recommendations from the capability inventory (see
  AGENTS_SKILLS_TOOLS.txt) into a phased, ordered plan. Execute one phase at a
  time; do not merge phases unless dependencies are satisfied.

Principles
  • Small, reviewable changes; no big-bang refactors unless a phase says so.
  • Prefer automation that keeps docs honest over one-off manual edits.
  • Security and policy docs should match how tools are actually named in the
    registry (native vs pluginId.tool vs mcp_*__*).

Out of scope for this plan (unless you open a new plan)
  • Renaming existing tools or breaking plugin IDs.
  • Replacing AgentFactory.choose() with an ML classifier (optional future).
